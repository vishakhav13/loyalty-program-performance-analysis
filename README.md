# Loyalty Program Performance & Incrementality Case Simulation

## Business Objective
Evaluate whether a loyalty program is driving incremental revenue or simply reflecting existing customer behavior.

## Analytical Approach
- Participation and revenue penetration analysis
- Basket value evaluation
- Incrementality assessment
- Cohort-based framework proposal
- Pre vs post enrollment analysis design

## Key Findings
- 80% loyalty participation rate
- Revenue contribution proportional to participation
- No observable incremental lift at aggregate level

## Strategic Recommendation
Enable customer-level linkage and implement causal measurement framework to quantify true ROI and retention impact.

## Tools Used
- Excel modeling
- Business analytics framework
